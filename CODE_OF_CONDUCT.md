# Contributor Covenant Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/).

Summary:
- Be respectful and kind.
- No harassment or discriminatory language.
- Report issues to the maintainers if you experience unacceptable behavior.

For the full text, see: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

Maintainers may remove, edit, or reject contributions that do not follow the code of conduct.
